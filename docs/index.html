SASSET - Smart Asset Management

What is SASSET?

SASSET is a library-based asset management software suite that can be tailored to virtually any type of assets to be managed. Anything from Infrastructure components, to your personal To-Do list at home.

SASSET Libraries

  1. sasset-core - This is the only library that will be required, regardless of how SASSET is to be installed and configured.
  2. sasset-rest - This provides a RESTful interface to SASSET, which is also required by some other libraries such as sasset-cli and sasset-www.
  3. sasset-cli - A command line utility which can be used to create/read/update/delete assets, just as you would via the UI or the RESTful interface
  4. sasset-www - Web interface for SASSET

Package Functionality

  • Database interaction, through the MongooseJS package
  • Account/Group/Role/Permissions management
  • Asset management
  • Partition/Field management
  • Backups
  • Authentication (JWT Tokens, the actual HTTP Basic auth is done in a different package, and the JWT token is stored via a cookie, in the same package as the HTTP auth)
  • Plugin oriented design
