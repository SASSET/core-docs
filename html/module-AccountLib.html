(static) getAccount(attr, nice) → {object|Mixed|undefined}

Source:
  • lib/account.js, line 66
Retrieve account data, either full object, or a specific attribute
Parameters:
Name Type Description
attr string Specific attribute to return
nice boolean If set to false, then instead of returning undefined for unset account data, an error is thrown. (Nice = true by default, needs to be set to false)
Returns:
If no attr is defined, full accountData object is returned, if attr is defined, then just that attribute is returned (mixed), if there is no account data set, then undefined is returned (or error thrown if nice is set to false
Type
object | Mixed | undefined
